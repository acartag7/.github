# Arnold Cartagena

Building **[Edictum](https://github.com/edictum-ai/edictum)** — runtime governance for AI agents. Contracts enforce what tools can do, before they execute.

Platform engineer turned founder. Based in Bern, Switzerland 🇨🇭

---

### What I'm building

[**Edictum**](https://github.com/edictum-ai/edictum) — deterministic YAML contracts on every agent tool call, outside the LLM. Python (⭐24), [TypeScript](https://github.com/edictum-ai/edictum-ts), and [Go](https://github.com/edictum-ai/edictum-go) SDKs. Self-hostable [operations console](https://github.com/edictum-ai/edictum-console) (⭐16) for HITL approvals, audit, and fleet monitoring.

### Selected work

- [**gap-benchmark**](https://github.com/acartag7/gap-benchmark) — Measuring text-safety vs tool-call-safety divergence across regulated domains.
